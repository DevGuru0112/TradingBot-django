# binance_gtm
No Investment Advice

## How to Run? ##
* For Developers
	* For Windows Users
		1. First create environment variables\
	   	  `$env:FLASK_APP="gtm"`\
	   	  `$env:FLASK_ENV="development"`\
	  	  `$env:PYTHONDONTWRITEBYTECODE=1` (Ignore pyc file)

		2. Run App\
	  	  `flask run`
	* For Mac Users
		1. First create environment variables\
		  `$env:FLASK_APP="gtm"`\
	   	  `$env:FLASK_ENV="development"`\
		  `$env:PYTHONDONTWRITEBYTECODE=1` (Ignore pyc file)
		
	        2. Run App\
		  `flask run`
